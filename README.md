# [Prime-Lime]

## What is Prime-Lime?
`Prime-Lime`は`Firebase`のHosting機能と`Rust`のWebフレームワークである`yew`の練習用に制作されたWebアプリケーションです。

## What Can Prime-Lime Do?
`Prime-Lime`では**第n項目の素数を算出する**ことができます。

バックエンドにはアトキンの篩のアルゴリズムを基に制作された`Primal`という`Rust`の外部クレートを使用しています。
